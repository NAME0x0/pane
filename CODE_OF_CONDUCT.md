# Code of Conduct

## Our Standard

Be respectful. Be constructive. Assume good intent.

We're building something useful and everyone is welcome — regardless of experience level, background, or identity.

## Expected Behavior

- Be kind and considerate in communication
- Respect differing viewpoints and experiences
- Accept constructive feedback gracefully
- Focus on what's best for the project and community

## Unacceptable Behavior

- Harassment, insults, or derogatory comments
- Personal or political attacks
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject any contributions not aligned with this code of conduct. Repeat offenders may be banned from the project.

## Reporting

If you experience or witness unacceptable behavior, open an issue or contact the maintainers directly.

This code of conduct applies to all project spaces — issues, pull requests, discussions, and any other communication channels.
